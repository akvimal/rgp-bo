Filing Instructions
  1. Verify all sales and purchase data is correctly captured
  2. Reconcile ITC with GSTR-2B data from GST portal
  3. Calculate net tax payable (Output Tax - ITC)
  4. Make payment through GST portal (Challan PMT-06)
  5. File GSTR-3B return on GST portal by 20th of next month
  6. Save acknowledgment and payment receipts
Tip: File GSTR-1 first (by 11th), then reconcile with GSTR-2B, and finally file GSTR-3B with payment (by 20th).
