What happens next?
  • A new payroll run will be created in DRAFT status
  • The period dates will be automatically set (1st to last day of the selected month)
  • You can then calculate payroll for all employees or specific employees
  • After calculation, you can review and approve the payroll
  • Finally, you can request payment processing for approved payroll
Tips
  • Check for existing runs: Ensure you don't have an active payroll run for the same period
  • Employee salary structures: Make sure all active employees have valid salary structures configured
  • KPI scores: For better incentive calculations, ensure KPI scores are updated for the month
  • Leave and attendance: Update leave requests and attendance records before calculating payroll
