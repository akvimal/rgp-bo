# Documentation

This folder contains comprehensive documentation for the RGP Back Office project.

## Structure

### `/analysis` - Analysis & Design Documents
- **POS_UI_ANALYSIS_AND_AI_RECOMMENDATIONS.md** - Comprehensive UI/UX analysis and AI integration opportunities for the POS system
- **POS_SCREENSHOT_SPECIFIC_IMPROVEMENTS.md** - Screenshot-based targeted improvements with implementation details

### `/planning` - Project Planning & Implementation Tracking
- **IMPROVEMENT_PLAN.md** - Overall project improvement roadmap
- **PHASES_1-3_COMPLETION_SUMMARY.md** - Summary of completed phases 1-3
- **PHASE4_ERROR_HANDLING_GUIDE.md** - Phase 4 error handling implementation guide
- **PHASE4_INFRASTRUCTURE_COMPLETE.md** - Phase 4 completion documentation
- **QUICKSTART_FIXES.md** - Quick reference for common fixes
- **TESTING_CHECKLIST.md** - QA testing checklist
- **CUSTOMER_HISTORY_ISSUES.md** - Customer history feature bug tracking and fixes
- **MANUAL_VERIFICATION_GUIDE.md** - Manual testing procedures
- **VERIFICATION_RESULTS.md** - Test verification results

### Root Level
- **READY_FOR_PULL_REQUEST.md** - Pull request readiness checklist

## Related Folders

- **/.github/** - GitHub templates (PR templates, issue templates, workflows)
- **/sql/migrations/** - Database migration scripts and guides
- **/api-v2/test/** - E2E test specifications
- **/.claude/** - Claude Code configuration
